File Format Requirements

Your Excel (.xlsx or .xls) file must contain the following columns. The order does not matter.

  • Name: The name of the specification (e.g., "Color", "Weight (kg)").
  • Type: Must be one of: text, numeric, choice.
  • Choices: For choice type only. A comma-separated list of options (e.g., "Red,Green,Blue").
  • Min Value: For numeric type only. The minimum allowed numeric value.
  • Max Value: For numeric type only. The maximum allowed numeric value.
Download Template Using the template is highly recommended.

Example:

Name Type Choices Min Value Max Value
Color choice Red,Green,Blue
Weight (kg) numeric 1 100
Temperature (°C) numeric -20 150
Serial Number text
